# Finance-Tracker

A full-stack personal finance management application built with Flask, MySQL, HTML, CSS, and JavaScript.

## Features

### Authentication

* User Registration
* User Login
* User Logout
* Password Hashing

### Transactions

* Add Transaction
* Edit Transaction
* Delete Transaction

### Budget Management

* Monthly Budget Tracking
* Remaining Budget Calculation
* Budget Alerts

### Analytics

* Monthly Expense Chart
* Category Expense Pie Chart
* Dashboard Statistics

### Search & Filters

* Search Transactions
* Sort Transactions
* Category Filter
* Type Filter

### Export

* CSV Export
* PDF Export

## Technology Stack

Backend:

* Python
* Flask
* MySQL

Frontend:

* HTML
* CSS
* JavaScript
* Chart.js

## Installation

1. Clone repository

git clone <repo-url>

2. Create virtual environment

python -m venv .venv

3. Activate environment

4. Install requirements

pip install -r requirements.txt

5. Run project

python app.py
